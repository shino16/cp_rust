# cpr

Rust 競プロライブラリ

`src/main.rs` を置くと実行されます（`Cargo.toml` の `[[bin]]` 参照）。

[`shino16/cargo-auto-bundle`](https://github.com/shino16/cargo-auto-bundle) と使うことを想定しています。

`rustc` のバージョンは `1.42.0` 想定です。外部クレート依存はありません。
